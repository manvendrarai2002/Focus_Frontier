# User Guide - Focus Frontier 🧠

Welcome to Focus Frontier, your personalized brain-training platform! This guide will help you get started and make the most of your cognitive training.

## 🚀 Getting Started

### Creating an Account

1. **Navigate to the Homepage**
   - Open Focus Frontier in your web browser
   - You'll see the main hub with all available games

2. **Click "Login / Register"** in the top right corner

3. **Register**
   - Enter your display name
   - Provide a valid email address
   - Create a strong password
   - Click "Register"

4. **Login**
   - Enter your email and password
   - Click "Login"
   - You're now ready to start training!

### Playing Without an Account

You can play all games without registering, but:
- ❌ Your progress won't be saved to the cloud
- ❌ You won't see analytics and trends
- ❌ High scores will only save locally
- ✅ All games are fully playable

## 🎮 The Games

### 1. Memory Matrix 🧩

**Skill Focus**: Working Memory

**How to Play**:
1. Watch as tiles light up in sequence
2. Wait for the sequence to finish
3. Click the tiles in the SAME ORDER
4. The sequence gets longer each round
5. You have limited lives - each mistake costs one!

**Tips**:
- **Beginner**: 4x4 grid, slower pace, 4 lives
- **Intermediate**: 5x5 grid, moderate pace, 3 lives
- **Advanced**: 6x6 grid, fast pace, 2 lives
- Focus on the pattern, not individual tiles
- Use visual chunking (group tiles together mentally)
- The numbers show the order during playback

**Scoring**: Your score is the round number you reach

---

### 2. Reflex Runner 🏃

**Skill Focus**: Attention & Inhibition Control

**How to Play**:
1. A runner automatically moves forward
2. Obstacles approach from the right
3. Click anywhere or press SPACE to jump
4. Avoid hitting obstacles
5. Game speeds up as you score more points

**Tips**:
- **Beginner**: Slower obstacles, speed 1.8x
- **Intermediate**: Moderate speed, speed 2.0x
- **Advanced**: Fast obstacles, speed 2.4x
- Time your jumps - don't jump too early!
- Watch the obstacle's position, not just the runner
- Build streaks for milestone celebrations

**Scoring**: 1 point per obstacle avoided

---

### 3. Color Cascade 🌈

**Skill Focus**: Attention & Inhibition (Stroop Effect)

**How to Play**:
1. A word appears (e.g., "RED")
2. The word is colored differently (e.g., blue)
3. Click the button matching the COLOR, NOT the word
4. Score points for correct answers
5. Lose time for wrong answers
6. Game ends when timer reaches 0

**Tips**:
- **Beginner**: 60 seconds
- **Intermediate**: 55 seconds
- **Advanced**: 45 seconds
- Read the word, but respond to the COLOR
- This tests your ability to ignore distractions
- Correct = +2 points, Wrong = -1 point & -2 seconds

**Scoring**: Points based on correct responses

---

### 4. Pattern Path 🛤️

**Skill Focus**: Cognitive Flexibility & Working Memory

**How to Play**:
1. Watch as tiles light up showing a path
2. The path expands each round (connects to neighbors)
3. Click tiles to trace the EXACT path shown
4. Complete before the timer runs out
5. Wrong tile = game over

**Tips**:
- **Beginner**: 14 seconds per round
- **Intermediate**: 12 seconds per round
- **Advanced**: 8 seconds per round
- The path always connects to adjacent tiles
- Visualize the shape the path makes
- Start clicking as soon as playback ends

**Scoring**: Round number reached

---

### 5. Shape Sorter 📦

**Skill Focus**: Processing Speed

**How to Play**:
1. Shapes fall from the top
2. Click each shape before it reaches the bottom
3. Score points for each shape caught
4. Miss a shape = no points but game continues
5. Speed increases as you score more

**Tips**:
- **Beginner**: Spawn rate 1800ms, fall speed slow
- **Intermediate**: Spawn rate 1500ms, fall speed medium
- **Advanced**: Spawn rate 1200ms, fall speed fast
- Scan the whole screen constantly
- Click shapes early rather than waiting
- Success rate decreases the spawn rate

**Scoring**: 1 point per shape clicked

---

### 6. Focus Sphere 🎯

**Skill Focus**: Attention & Reaction Time

**How to Play**:
1. A target sphere appears randomly on screen
2. Click the sphere before it disappears
3. Score increases with each hit
4. Score decreases slowly over time
5. Missed clicks reduce your score

**Tips**:
- **Beginner**: 1200ms window
- **Intermediate**: 1000ms window
- **Advanced**: 800ms window
- Keep your cursor/finger ready
- Anticipate the sphere's next position
- Hit = +10 points, Miss = -3 points
- Building streaks speeds up the game

**Scoring**: Points based on hits

## 📊 Understanding Your Analytics

### Accessing Analytics

1. Click "Open analytics" on the homepage
2. Or navigate to the Analytics page from any game
3. Login required to see your personal stats

### What You'll See

**Per-Game Performance**
- **Best Score**: Your highest achievement in each game
- **Average Score**: Mean score across all sessions
- **Plays**: Total number of sessions

**Trend Chart**
- Daily average scores over time
- Helps you see improvement or patterns
- Rolling 30-day window

**Skill Buckets**
- Performance by cognitive skill
- Average scores per skill area:
  - Working Memory
  - Attention
  - Inhibition
  - Cognitive Flexibility
  - Processing Speed

**Recommendations**
- AI-generated suggestions based on your weakest skills
- Focus on these areas for balanced improvement

## ⚙️ Settings & Preferences

### Sound Control

**Toggle Sound Effects**:
- Click the "Sound ON/OFF" button in any game
- Your preference saves across all games
- Re-enable anytime

### Accessibility Features

**High Contrast Mode**:
- Better visibility for users with vision needs
- Toggle in settings (coming soon to UI)
- Or set in console: `localStorage.setItem('pref_highContrast', '1')`

**Font Scaling**:
- Increase or decrease text size
- Set in console: `localStorage.setItem('pref_fontScale', '1.2')`
- Values: 0.8 (smaller) to 1.5 (larger)

**Reduced Motion**:
- Minimize animations for motion sensitivity
- Set in console: `localStorage.setItem('pref_reducedMotion', '1')`

### Managing Your Data

**View Local Stats**:
- Open browser console (F12)
- Type: `localStorage`
- See all saved preferences and stats

**Clear Local Data**:
- Open browser console
- Type: `localStorage.clear()`
- This resets tutorials, sounds, and local high scores
- Server data (sessions) remains intact if logged in

**Export Your Data**:
- Go to Analytics page
- Sessions are stored on server if logged in
- Contact support to request data export

## 🎯 Tips for Effective Training

### General Guidelines

1. **Consistency Over Intensity**
   - Play 10-15 minutes daily rather than 1 hour weekly
   - Regular practice builds neural pathways

2. **Start at Your Level**
   - Don't jump to Advanced immediately
   - Master Beginner before moving up
   - Challenge yourself, but don't frustrate yourself

3. **Focus on Weak Skills**
   - Check your Analytics regularly
   - Prioritize games targeting your lowest skill scores
   - Balanced improvement is key

4. **Track Progress**
   - Review analytics weekly
   - Celebrate small improvements
   - Trends matter more than daily fluctuations

5. **Take Breaks**
   - Rest between sessions
   - Cognitive fatigue reduces effectiveness
   - Fresh mind = better performance

### Training Plans

**Beginner Plan (Week 1-2)**:
- Memory Matrix: 2x daily (5 min each)
- Focus Sphere: 1x daily (5 min)
- Track high scores and try to beat them

**Intermediate Plan (Week 3-4)**:
- Rotate through all 6 games daily
- 5 minutes per game
- Increase difficulty when comfortable

**Advanced Plan (Week 5+)**:
- Focus on analytics recommendations
- Target weakest skills for 70% of time
- Maintain strong skills for 30% of time

## 🆘 Troubleshooting

**Game won't load**:
- Refresh the page (F5)
- Clear browser cache
- Try a different browser
- Check internet connection

**Can't login**:
- Verify email and password
- Check for typos
- Try password reset (if available)
- Clear cookies and try again

**Scores not saving**:
- Ensure you're logged in
- Check internet connection
- Look for error messages
- Try registering/logging in again

**Sound not working**:
- Check browser permissions
- Unmute tab/browser
- Try another browser
- Toggle sound off and on

**Analytics not showing**:
- Login required
- Play a few games first
- Refresh the analytics page
- Allow 24 hours for data to populate
